The MAX Language offers complete control of the Synclavier (R) II Digital Synthesizer and access 
to all internal computer capabilities. lt consists of a library of special XPL/4 procedures which 
you can use in the development of your own customized digital synthesis system. 
All New England Digital hardware is supported.

With MAX, you an create a system that allows a performer to interact with the Digital Synthesizer in new ways. 
You are not restricted to the Synclavier (R) II real-time system. The pedal could be used to control recording speed. 
Or, input from the Keyboard, pedal, and knob could direct the computer to generate a particular set of pitches in a 
unique manner.

From: MAX LANGUAGE USER GUIDE June 15, 1982
